# FAAH - Sound on Failure

FAAH is a VS Code extension that plays the iconic FAAH sound whenever your tests, builds, or programs fail in the terminal.

## Features

- Plays the FAAH sound when your tests, builds, or programs fail in the terminal.
- Shows a random picture from the failure-images folder when an error is detected.
- Supports all major programming languages and build tools.
- Supports all major testing frameworks.
- Supports all major runtime environments.

## Installation

download the latest release from the [releases](https://github.com/titus7490/faah-cursor-extension/releases) page and install it in VS Code. (right click on the downloaded file and select "Install Extension")

## Usage

The extension will automatically play the FAAH sound when your tests, builds, or programs fail in the terminal.

## Configuration

You can configure the extension in the VSCode settings. There you can enable/disable the sound and the picture. You can also configure the sound and the picture to be played when certain events occur.
